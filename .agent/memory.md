# CareerCoachAntigravity Project Memory

## Project Identity & Values
- **Aesthetic North Star**: The project demands a premium, "wow-factor" design. This includes vibrant colors, dark mode by default, glassmorphism, and smooth micro-animations. No placeholders or generic layouts are allowed.
- **Goal**: To build a world-class AI Career Intelligence platform that helps developers navigate the modern AI job market.

## Tech Stack & Architecture
- **Framework**: Next.js 14+ (App Router).
- **Languages**: TypeScript (Strict mode).
- **Styling**: Vanilla CSS for maximum control, combined with modern CSS variables.
- **Brain**: Powered by a "Dual-Brain" setup:
    - **Antigravity Interface**: Used for UI development, deep web research via sub-agents, and high-level architectural planning.
    - **Claude Code CLI**: Used for rapid terminal operations, Git management, and test execution.

## User Preferences
- **Testing**: Use Vitest for library logic.
- **Workflow**: Leverage slash commands in `.agent/workflows/` for recurring tasks.
- **Installation**: Claude Code CLI 2.1.2 is installed at `~/.local/bin/claude` and in the PATH.

## Active Missions
1. **Career Intelligence System**: Implementing Skill Graph Navigator using **Ternary Relationships** (User/Position/Company) and **Temporal Knowledge Graphs**.
2. **Resume Builder**: Advanced matching and profile parsing, incorporating **CareerScape** integrity signatures.
3. **Interview Prep**: Conversational simulator with scoring.
